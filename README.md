# QA/PM Board

Jira에 없는 QA/PM 특화 기능을 제공하는 통합 일정 관리 웹 플랫폼입니다.

## 주요 기능

### 보드
- **칸반** - 드래그&드롭 상태 변경
- **타임라인** - 간트 차트 시각화
- **캘린더** - 월별 일정 뷰
- **테이블** - 정렬/필터 가능한 리스트

### 테스트 관리
- **테스트 케이스** - 스텝/기대결과 구조화, 실행 기록 (Pass/Fail/Skip + 스크린샷)
- **커버리지 매트릭스** - 기능 영역별 테스트 커버리지 시각화
- **회귀 테스트 세트** - 릴리스별 테스트 묶음 관리

### QA
- **재현율 추적** - 간헐적 버그의 재현 횟수/조건 기록
- **디바이스 매트릭스** - OS/브라우저/디바이스별 테스트 현황
- **릴리스 체크리스트** - 배포 전 QA 사인오프 (템플릿 적용 가능)
- **결함 밀도 대시보드** - 모듈/스프린트/우선순위별 버그 차트

### PM
- **릴리스 캘린더** - QA/개발/배포 일정 오버레이
- **리스크 히트맵** - 마감 초과, 버그 집중, 테스트 부족 영역 시각화
- **QA 워크로드** - 담당자별 업무 부하 차트
- **테스트 진행률** - 실행률, Pass Rate, 번다운 차트

### 편의 기능
- **일일 리포트** - 마크다운 자동 생성 + 클립보드 복사
- **스크린샷 주석** - Canvas 기반 이미지 마킹
- **체크리스트 템플릿** - 반복 QA 절차 템플릿화

## 기술 스택

- **Frontend**: React 19 + Vite
- **차트**: Recharts
- **아이콘**: Lucide React
- **저장소**: localStorage (Supabase/Firebase 전환 가능)

## 실행

```bash
npm install
npm run dev
```

## 프로젝트 구조

```
src/
├── constants/        # 타입, 상태, 네비게이션 정의
├── utils/            # 날짜, ID 헬퍼
├── store/            # Context + useReducer 상태관리
├── components/
│   ├── layout/       # 사이드바, 헤더, 필터바
│   ├── shared/       # 공용 컴포넌트
│   ├── views/        # 각 기능별 뷰
│   └── forms/        # 생성/편집 폼
└── styles/           # CSS 변수, 글로벌 스타일
```
